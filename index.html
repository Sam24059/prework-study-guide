HTML

  • The head element contains information about the webpage.
  • The head element is not visible to users.
  • The bodty element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A paddding indictates how much space we want around the content inside an element.
  • Inline-css: Style element directly in HTML. Quick and permanent, less flexible. Must be edited individually.
  • Internal-css: Embed entire style sheet directly into HTML using style in head element.
  • External css: Most common. Allows developers to keep all CSS rules in seperate file. Use link element in head.

Git

  • git status: checks what branch we are currently working on
  • pwd: prints working directory
  • ls: shows list working directories
  • cd: changes directory
  • cd ..: moves back directory
  • git checkout -b branch-name: creates new branch and switches to it
